## How to use

1. help : 어떤 Command가 있는지 알려줘요.
2. 지역코드 리스트 : 지역별로 어떤 코드인가요? 
3. 지역코드 검색 : 지역코드로 검색하면 어떤 상영관이 있는지 알 수 있어요.
4. 상영관코드 리스트 : 상영관코드를 알아야 영화검색이 가능해요.
5. 상영관검색 : 한글로도 상영관코드를 알아낼 수 있어요.
6. 날짜검색 : 날짜로 내 상영관에서 하는 영화가 무엇이 있는지 알아볼까요?
7. 무비차트 : 실시간 인기영화는 !!!
8. 알람 / 설정 : 원하는 날짜, 원하는 영화, 원하는 상영관으로 알람 설정하기
9. 알람 / 조회 : 내가 설정했던 알람은 뭐가 있더라?
10. 알람 / 삭제 : 그 영화 포기에요. 안볼래요.
